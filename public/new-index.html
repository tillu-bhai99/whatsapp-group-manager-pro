How to Find Your WhatsApp Group ID
Easiest Method: Invitation Link

Simply paste a WhatsApp invitation link and the system will automatically handle everything - no group ID needed!

To get a group invitation link:
  1. Open the WhatsApp group
  2. Tap the group name at the top
  3. Scroll down and tap "Invite to Group via Link"
  4. Copy the link
  5. Paste the entire invitation link into the field

Example link: https://chat.whatsapp.com/AbCdEfGhIjKl

Advanced: Using Group ID

If you're already a member of the group, you can use the group ID directly.

A valid WhatsApp group ID looks like: 1234567890-1234567890@g.us or 123456789012345@g.us

Common Issues & Solutions

If the QR code is not displaying:

  1. Click the "Refresh QR Code" button
  2. Check if the server is running on the correct port
  3. Run the "Socket Connection Test" from the diagnostics menu
  4. Try opening the app in a different browser

If you're having trouble connecting to WhatsApp:

  1. Make sure your phone has an active internet connection
  2. Verify that WhatsApp is installed and working on your phone
  3. Completely close WhatsApp on your phone, then reopen and try again
  4. Try using the "QR Code Test" from the diagnostics menu
